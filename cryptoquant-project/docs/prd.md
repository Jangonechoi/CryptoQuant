1. Detailed Product Description

1.1 제품 개요

본 제품은 암호화폐 시세 데이터를 시각적으로 제공하고,
사용자가 자동매매 전략을 설정하여 모의 백테스트를 수행할 수 있는 웹 기반 데모 플랫폼이다.

외부 암호화폐 데이터 API를 활용하여 실시간 또는 과거 시세를 조회하고,
차트 기반 UI를 통해 가격 흐름과 전략 결과를 직관적으로 확인할 수 있도록 설계된다.

또한 소셜 로그인, 결제(구독) 구조를 포함하여
실제 상용 SaaS 서비스와 유사한 사용자 흐름을 제공하는 것을 목표로 한다.

⚠️ 본 서비스는 실제 거래를 실행하지 않으며,
모든 자동매매 기능은 학습 및 시뮬레이션 목적으로만 제공된다.

⸻

1.2 제품 목표
	•	암호화폐 시세 데이터 활용 경험 제공
	•	백엔드(FastAPI)와 프론트엔드(React) 간 역할 분리 구조 학습
	•	차트 기반 데이터 시각화 UX 구현
	•	결제, 구독, 인증을 포함한 실서비스 구조 경험
	•	자동매매 전략 및 백테스트 로직에 대한 기초 이해

⸻

1.3 문제 정의

문제	설명
암호화폐 분석 진입 장벽	초보자에게 암호화폐 차트와 지표는 직관적이지 않음
자동매매 학습 환경 부족	실거래 이전에 안전하게 전략을 실험할 공간이 부족
서비스 흐름 이해 부족	단순 차트 앱이 아닌, 실제 서비스 구조를 경험하기 어려움


⸻

1.4 해결 방안
	•	차트 중심의 UI로 시세 흐름을 직관적으로 제공
	•	실제 거래가 아닌 모의 자동매매 & 백테스트 기능 제공
	•	소셜 로그인과 구독 기능을 통해 현실적인 SaaS 플로우 구현

3. Core Features and Specifications

3.1 암호화폐 시세 조회 및 차트 제공

기능 설명
	•	주요 암호화폐(BTC, ETH 등) 시세 데이터 조회
	•	캔들 차트 기반 가격 시각화
	•	기간 선택 (1분, 5분, 1시간, 1일 등)

주요 UX 요소
	•	인터랙티브 차트 (확대/축소, 툴팁)
	•	이동평균선(SMA, EMA) 표시

⸻

3.2 회원 / 소셜 로그인

기능 설명
	•	소셜 로그인 기반 사용자 인증
	•	로그인 상태 유지

기대 효과
	•	회원가입 마찰 최소화
	•	개인별 전략 및 기록 관리 가능

⸻

3.3 PortOne 결제 기능

기능 설명
	•	구독형 결제 구조
	•	무료 / 프리미엄 플랜 구분

프리미엄 기능 예시
	•	백테스트 기간 확장
	•	전략 파라미터 조정
	•	상세 성과 리포트 열람

결제 대상은 투자 행위가 아닌 서비스 기능 접근 권한임

⸻

3.4 자동매매(모의) & 백테스트

기능 설명
	•	전략 기반 매수/매도 시뮬레이션
	•	과거 데이터 기반 백테스트 실행

제공 정보
	•	누적 수익률
	•	거래 횟수
	•	손익 추이
	•	최대 낙폭(MDD)

⚠️ 실거래 API 연동 및 실제 주문 실행 기능은 제공하지 않음

⸻

3.5 마이페이지 / 대시보드

기능 설명
	•	사용자 정보 조회
	•	구독 상태 및 결제 내역 확인
	•	전략 실행 이력 확인

	3.	백테스트 실행
	4.	결과 차트 및 성과 확인

	1.	프리미엄 기능 클릭
	2.	결제 페이지 이동
	3.	PortOne 결제 완료
	4.	제한 해제된 기능 사용
